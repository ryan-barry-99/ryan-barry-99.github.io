Air Hockey Robot

A custom arm robot designed to play Air Hockey against a human opponent via object detection and tracking from an overhead camera.

Robot in Action

Here the robot defends its goal.

The reason it misses a puck on the right is because we had the workspace limits intentionally narrower than the table to avoid robot suicide :)

  • Collected 26,627 images during gameplay; labeled 443 images (64 challenging).
  • Processed images for puck and line detection using YOLOv8; cleaned to 15,763 images.
  • Augmented dataset by horizontally flipping the table for a total of 31,525 images.

Used YOLOv8 to label bounding boxes for the puck and black lines on the table, forming the foundation for physics calculations and predictions.

Developed a physics-based predictive model to determine puck trajectory and reflection points:

If dx ≈ 0: Yc = 0.5, Θ = 0
elif dx > 0:
    Calculate bounce at Y = 1; reverse dy
else:
    Calculate bounce at Y = 0; reverse dy

Leveraged a Long Short-Term Memory (LSTM) neural network to predict puck trajectory based on dynamic movement:

  • MSE: 0.0573
  • Missed Shots: 25/135
  • 3 DoF planar robot
  • SSC-32U PWM servo driver
  • Custom kinematics library using Denavit–Hartenberg parameters
  • Inverse kinematics lookup table for rapid position adjustments
